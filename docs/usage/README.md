## Installation

### パッケージのみインストールする場合
```
$ python -m pip install git+https://github.com/jjj999/bamboo.git
```

### サーバー上でビルドする場合
```
$ git clone https://github.com/jjj999/bamboo.git
$ cd bamboo
$ sudo python3 build.py install
```
